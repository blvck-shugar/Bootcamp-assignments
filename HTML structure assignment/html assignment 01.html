Humans are pattern-matching machines, to solve problems we draw upon previous experiences to identify possible solutions. Languages were developed from matching patterns and the human innate desire to relate with one another. Programming languages are not so different, they are another form language mankind has invented only difference is, it was not made to relate to people but to relate to machines. A programming language is a set of instruction, syntax, and commands used to create a software program. Programmers use high level languages to write codes, this code can be compiled into low level language which the computer hardware recognizes. High level languages are used in most major programming languages and are designed to be easy to read and understand, this allows programmers to write source codes in natural fashion. High level languages include C++, PHP and Java. Low level languages include assembly and machine language, an assembly language contains a list of basic instructions and is much more difficult to read than a high-level language
